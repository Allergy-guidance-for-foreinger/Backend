Google ID Token 인증 테스트

이 페이지는 프론트 없이 실제 Google ID token을 발급받고, 현재 서버의 /auth/login, /auth/refresh, /auth/logout 흐름을 확인하기 위한 로컬 테스트 페이지입니다.

1. Google 로그인 설정

서버에서 읽는 중...

Google Client ID는 서버 설정값에서 자동으로 읽어옵니다. Google Cloud Console의 승인된 JavaScript 원본에는 현재 페이지 주소가 포함되어 있어야 합니다. 예: http://localhost:8080

2. 서버 로그인 호출

먼저 Google 로그인으로 실제 ID token을 받은 뒤 호출해야 합니다.

3. 토큰 재발급 / 로그아웃

로그아웃은 현재 access token을 Authorization 헤더에 넣고, refresh token을 요청 본문에 넣어 호출합니다.

현재 상태

아직 값이 없습니다.
